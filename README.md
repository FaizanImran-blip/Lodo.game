## 🎯 Game Overview

> A console version of Ludo where 4 players (Red, Green, Blue, Yellow) compete to complete their 4 tokens first.

- 🎲 Dice roll mechanic
- 🔁 Turn-based logic
- 🚩 Win condition (first to complete 4 tokens)
- ❌ Loser detection (least completed tokens)
